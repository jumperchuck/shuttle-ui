Button

Import​

import { Button } from '@shuttle-ui/components';
  • Button
  • Button.Group

Examples​

Basic​

传递type参数改变形式，有三种形式：实心（默认值）、描边、文本

Color​

传递color参数来自定义颜色，可以是theme配置中的颜色

Size​

传递size参数来设置组件大小，可以通过theme来扩展修改此参数

Loading​

传递loading参数显示加载状态并且禁止点击

Icon​

传递icon参数渲染图标

Group​

通过Button.Group包裹的按钮，可以传递buttonProps、selectedKeys、buttonProps参数处理
